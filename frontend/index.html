Two-Pass Architecture

Our intelligent pipeline uses a two-pass strategy for maximum accuracy.

📄

Input Document

Mixed PDF / Image with multiple document types

Pass 1
🔍

Classify & Segment

AI identifies document types and splits pages into logical segments

Pass 2
📊

Route & Extract

Each segment routed to a specialized extractor for high-accuracy field extraction

✅

Structured Output

JSON + human-readable fields with confidence scores

⚡

Parallel Execution

Different analyzers run simultaneously via ThreadPoolExecutor for ~3× speedup

🎯

Page-Level Routing

Only relevant pages sent to each analyzer via content_range — no wasted API calls

🔀

Virtual Segments

Multi-category pages (e.g., PAN + Aadhaar on one image) are automatically split
